# 카페 프로젝트 - 스피드 버전

## 오프닝 (5초)
"카페 주문 웹앱, '카페' 프로젝트입니다."

## 1. 기획 의도 (15초)
- 첫 프로젝트, 프론트엔드 기초 다지려고 바닐라 HTML/CSS/JS로 직접 구현
- 같은 카페 앱이라도 "다르게" 만들고 싶어서 유럽풍 부티크 컨셉으로 전체 리디자인
- 차별화: **나만의 레시피 저장 + 원클릭 재주문**

## 2. 기술스택 (10초)
- HTML/CSS/JS, localStorage, 회원가입/로그인, Git, **GitHub Pages 배포**

## 3. 시연 (라이브, 말 짧게 — 클릭하면서 한 단어씩)
1. "메인 페이지 — 배포 링크로 접속"
2. "메뉴 → 옵션 선택 → 장바구니 미리보기"
3. "레시피 저장"
4. "장바구니 → 주문"
5. "마이페이지 — 즐겨찾기, 레시피"
6. "회원가입/로그인"
7. "관리자 페이지 — 로그인 가드"

## 4. 트러블슈팅 (한 문장씩, 4개)
- "레시피 패널 넣으니 카드가 밀려서 → 분리해서 해결"
- "GitHub Pages 배포하니 이미지 다 깨져서 → 경로를 동적으로 계산해서 해결"
- "로그인 도입 후 데이터 공용이라서 → 계정별 분리"
- "관리자 페이지 뚫려있어서 → 로그인 가드 추가"

## 5. 개선점 (5초, 빠르게 나열)
"백엔드 연동, 결제, 모바일 UX, 이미지 자동화 — 이 네 가지가 다음 단계입니다."

## 클로징 (3초)
"이상입니다, 감사합니다."
